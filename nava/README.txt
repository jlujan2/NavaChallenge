Description
* The project takes two files from /data and /schema with the same name but different extension, process it and return
  the value has a hash map using the rest api

Instuction to run the application:
* Get STS from https://spring.io/tools
* Open STS 
	* File->Import-> Exsisting Maven Project
	* Browse the Project File->Finish
* Run the application
	* Right click on NavaAplication -> Run as -> Java Project

* Use the Rest Api	
	* Go to localhost/nava/{name of the file}
